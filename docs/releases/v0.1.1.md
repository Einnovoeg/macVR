# macVR 0.1.1

Release date: 2026-03-14

## Summary

`0.1.1` was the last cleanup-focused patch line before the `0.2.0` runtime expansion. It focused on repository hygiene, metadata consistency, and a small host runtime fix.

## Highlights

- Clean shutdown handling for `macvr-host` on `Ctrl+C` and termination signals
- README cleanup to remove personal URLs and keep installation instructions publication-ready
- Stronger attribution text in the release docs and third-party notices
- Version metadata tightened so the checked-in `VERSION` file drives the C sender release string
- JPEG quality clamp aligned with the documented `1-100` CLI contract
