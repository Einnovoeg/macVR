# macVR 0.1.0

Release date: 2026-03-13

## Summary

`0.1.0` was the initial public scaffold for the native macOS transport stack. It introduced the host, client, bridge ingest path, bridge simulator, and JPEG sender tooling.

## Highlights

- Native macOS host and client transport scaffold for `mock`, `display-jpeg`, and `bridge-jpeg` modes
- Authenticated bridge ingest service with TCP control and UDP chunk transport
- Bridge simulator reconnect logic, keepalive handling, and local TCP JPEG ingest
- Portable `macvr-jpeg-sender` helper for pushing JPEG frames into the bridge seam
- Initial public release metadata including versioning, license, dependencies, and release notes

## Historical Limitations In The 0.1.0 Line

These gaps were addressed in later releases:

- Experimental OpenXR runtime shim was not yet shipped
- Bundled native runtime service was not yet shipped
- Graphical control center was not yet shipped
