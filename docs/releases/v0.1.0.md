# macVR 0.1.0

Release date: 2026-03-13

## Summary

`0.1.0` is the first public cleanup and release-hardening pass for `macVR`. It packages the native macOS transport scaffold, bridge ingest path, reconnect fixes, and the portable JPEG sender into a releaseable repository layout with explicit licensing and documentation.

## Highlights

- Native macOS host/client transport path for mock, display, and bridge modes
- Token-authenticated bridge ingest with UDP chunk transport
- Bridge simulator reconnects, stale-frame handling, and local TCP JPEG input
- Portable JPEG sender for native and Wine/GPTK-side frame injection
- Public-release docs, changelog, license, notices, and dependency inventory

## Known Limitations

- No full OpenXR runtime is included
- No bundled headset runtime, SteamVR, Wine, Proton, ALVR, Virtual Desktop, or GPTK
- Current release is CLI-first and does not ship a graphical control center
